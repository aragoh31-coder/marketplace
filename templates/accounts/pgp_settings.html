üìö Getting Started with PGP

How to Generate a PGP Key

Command Line (GPG)

# Generate a new key pair
gpg --gen-key

# Export your public key
gpg --armor --export your-email@example.com

GUI Tools

  • Windows: Gpg4win (Kleopatra)
  • macOS: GPG Suite
  • Linux: Seahorse, KGpg
PGP Best Practices
  • Use a strong passphrase for your private key
  • Keep your private key secure and never share it
  • Create a revocation certificate in case your key is compromised
  • Use a key size of at least 2048 bits (4096 recommended)
  • Regularly backup your private key to a secure location
